**GNU Affero General Public License v3 (AGPL v3)**.

This repository contains the **AGPL-covered source code** used as part of
a larger system and includes modifications made to the original AzPixel
codebase to support specific workflows.

## Project Status

This repository is **not actively developed as a standalone product**.

The actively maintained service and extended features are provided
separately as part of a different system.

---

## Scope of This Repository

Included in this repository:
- AzPixel-based conversion scripts
- Modifications made under AGPL v3
- Core Java → Bedrock conversion logic

Not included in this repository:
- 3D icon generation system
- Animation processing engine
- Performance optimization layers
- Service infrastructure, automation, or business logic

The excluded components are developed and operated as **separate services**
and are **not derivative works of AzPixel**.
